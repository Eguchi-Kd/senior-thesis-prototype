# プロジェクト概要

## 研究タイトル
体験型ゲームを用いたデジタル詐欺認知学習の設計と効果検証

## 目的
スマートフォン世代（中高生〜若年層）を対象に、一人称視点3D探索ゲームを通じて  
**現実情報とデジタル情報の矛盾を発見する認知スキル**を訓練する。

単なる知識伝達ではなく「体験から気づく」プロセスを重視することで、  
未知の詐欺手口にも対応できる**転移可能な判断力**の習得を目指す。

## ゲームコンセプト
「8番出口」スタイルの一人称3D空間探索。  
プレイヤーは部屋を歩き回り、物理的環境（カレンダー、掲示物等）と  
デジタルデバイス上の情報（SMS、メール、QR等）の矛盾を見つけて判定する。

## ターゲットユーザー
- 中学生〜大学生（スマートフォンネイティブ世代）
- スマートフォン（Android/iOS）のブラウザでプレイ
